# Стиль кода

- Отступы: 2 пробела, кодировка UTF-8, перевод строки LF (`.editorconfig`).
- Форматирование: Prettier (`.prettierrc.json`).
- Линтинг: ESLint (`.eslintrc.json`).
- Именование: функции отрисовки — `renderX()`; данные — UPPER_CASE (`CATS`, `P`); состояние — `S`.
- Каждый модуль начинается с комментария о назначении.
