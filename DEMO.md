# Демонстрация приложения

## Файл `НК-ЗОЖ-демо.gif`
Анимированный обзор экранов приложения (отрисовка дизайна: онбординг → бюджет на сладкое →
каталог → карточка товара → дневник → корзина → заказ → админ-панель). Это наглядный
проход по интерфейсу. Для «живой» записи реальной работы — см. ниже.

## Как записать своё видео/GIF (Windows) — для защиты
- **Видео экрана:** нажми **Win + Alt + R** (встроенный Xbox Game Bar) — начнётся запись
  активного окна браузера. Останови той же комбинацией. Файл — в `Видео/Клипы`.
- **GIF:** поставь бесплатную программу **ScreenToGif** → «Запись» → выдели окно браузера →
  пройди сценарий ниже → «Сохранить как .gif».

## Сценарий для защиты (что показать по шагам)
1. **Запуск:** `npm run install:all` → `npm start` → открыть `http://localhost:3000`.
2. **Онбординг:** «Начать» → профиль (пол, возраст, рост, вес, цель) → «Рассчитать бюджет» →
   показывается дневной **бюджет на сладкое** (~20% суточной нормы по Миффлину–Сан Жеору).
3. **Каталог:** stories, баннеры, поиск, фильтры; открыть товар → КБЖУ, состав, отзывы → «В корзину».
4. **Дневник:** «+ Добавить в дневник» (поиск по каталогу или свой продукт) → кольцо бюджета;
   при переборе — мягкий нудж «Сегодня вы порадовали себя».
5. **Корзина и заказ:** промокод **ЗОЖ10**, списание бонусов → «Оформить заказ» → заказ оформлен.
6. **Аккаунт:** Профиль → «Войти» (`demo@nk.ru` / `demo1234`) → заказы и дневник синхронизируются с сервером.
7. **Админ-панель фабрики:** `http://localhost:3000/admin` (`admin@nk.ru` / `admin1234`) —
   список заказов, смена статуса (покупателю уходит уведомление), график продаж, пользователи.
8. **Документация API:** `http://localhost:3000/api/docs` (Swagger UI).

## Демонстрация бэкенда через консоль (curl)
```bash
# Регистрация
curl -X POST http://localhost:3000/api/auth/register -H "Content-Type: application/json" \
  -d "{\"email\":\"a@b.ru\",\"password\":\"secret123\",\"name\":\"Анна\"}"

# Каталог
curl http://localhost:3000/api/products

# Вход админом и список заказов
TOKEN=... # из ответа /api/auth/login для admin@nk.ru
curl http://localhost:3000/api/admin/orders -H "Authorization: Bearer $TOKEN"
```
